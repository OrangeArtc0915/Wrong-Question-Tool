# 错题整理工具 - 快速开始指南

## 🎯 5分钟快速上手

### 第一步：环境检查
```bash
# 检查Python版本（需要3.7+）
python --version

# 运行系统测试
python test.py
```

### 第二步：安装依赖
```bash
# Windows用户
install.bat

# macOS/Linux用户
./install.sh
```

### 第三步：启动程序
```bash
# Windows用户
start.bat

# macOS/Linux用户
./start.sh
```

## 📚 核心功能演示

### 1. 导入错题（30秒）
1. 点击"导入错题"按钮
2. 选择学科（如：数学）
3. 选择图片文件（支持多选）
4. ✅ 完成！文件自动分类保存

### 2. 浏览文件（10秒）
- 双击文件夹 → 进入子目录
- 双击文件 → 预览内容
- 右键文件 → 操作菜单

### 3. OCR识别（1分钟）
1. 选择图片文件
2. 点击"OCR识别"
3. 等待识别完成
4. 保存或复制文字结果

### 4. 导出文档（2分钟）
1. 点击"导出PDF"或"导出Word"
2. 选择导出范围
3. 选择保存位置
4. ✅ 完成！生成精美文档

## 🚀 性能优化建议

### 提升OCR速度
- 使用高分辨率图片
- 避免过大的图片文件
- 启用图片预处理

### 提升程序响应
- 定期清理临时文件
- 使用SSD存储
- 关闭不必要的后台程序

### 内存优化
- 避免同时处理过多文件
- 及时关闭不需要的预览
- 定期重启程序

## 🔍 故障排除

### 程序无法启动
```bash
# 重新安装依赖
pip install -r requirements.txt

# 检查Python版本
python --version

# 运行诊断测试
python test.py
```

### OCR识别失败
- 检查Tesseract是否安装
- 尝试图片预处理
- 确认图片格式支持

### 文件无法保存
- 检查目录权限
- 确认磁盘空间充足
- 以管理员身份运行

### 界面显示异常
- 更新显卡驱动
- 调整系统缩放设置
- 重启图形界面
