# 错题整理工具 - 更新日志

## v2.0.0 (2025-11-02)

### 🐛 BUG修复
- **修复图片保存BUG**: 解决了图片预处理后无法保存的问题
- **改进错误处理**: 增强了异常处理和用户提示
- **修复文件路径问题**: 改进了文件操作的安全性和稳定性

### ✨ 新功能
- **图片裁剪功能**: 
  - 支持自定义裁剪区域
  - 提供预设裁剪选项（正方形、顶部、中心）
  - 实时预览裁剪效果
- **图片旋转功能**: 
  - 支持90°、180°、270°旋转
  - 支持自定义角度旋转
- **标签管理系统**: 
  - 为文件添加自定义标签
  - 在文件列表中显示标签
  - 支持标签搜索和筛选
- **批量重命名**: 
  - 支持批量文件重命名
  - 可设置前缀、后缀、编号格式
  - 实时预览重命名效果
- **文件搜索功能**: 
  - 支持文件名搜索
  - 实时搜索结果
  - 搜索结果高亮显示
- **统计信息显示**: 
  - 显示总文件数、总大小
  - 按学科分类统计
  - 可视化统计图表

### 🎨 界面美化
- **主题样式系统**: 
  - 现代化的界面设计
  - 改进的颜色搭配
  - 统一的视觉风格
- **改进的布局**: 
  - 优化了工具栏布局
  - 添加了搜索框
  - 改进了状态栏信息
- **新的图标和按钮**: 
  - 美化的按钮样式
  - 改进的图标设计
  - 更好的视觉层次

### ⚡ 性能优化
- **快捷键支持**: 
  - Ctrl+I: 导入错题
  - Ctrl+R: 刷新列表
  - Ctrl+F: 搜索文件
  - F2: 重命名
  - Delete: 删除
- **自动备份**: 
  - 图片处理前自动备份原图
  - 可配置的备份选项
- **改进的文件管理**: 
  - 更快的文件列表加载
  - 优化的内存使用

### 🔧 工具增强
- **增强的右键菜单**: 
  - 根据文件类型显示相应操作
  - 统一的操作入口
- **批量操作**: 
  - 批量处理多个文件
  - 批量导出功能
- **设置系统**: 
  - 可配置的主题选项
  - 个性化设置保存

### 📱 跨平台改进
- **更好的兼容性**: 
  - 改进了不同操作系统的兼容性
  - 优化了字体显示
- **响应式设计**: 
  - 适应不同屏幕尺寸
  - 改进的布局弹性
